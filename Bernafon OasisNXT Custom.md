# How-To: Install custom variations of OasisNXT software

> Online version: https://github.com/JediLin/Hearing-Aids-Fitting-Software-Update-Checkers/wiki/Cust_Bernafon

1. Gets the Bernafon `setup.exe`, `Cust_Bernafon.msi`, and `Cust_XXXX.msi`
2. Run `setup.exe`
3. Right-click ¡÷ Uninstall on `Cust_Bernafon.msi`. Note: `setup.exe` installed this file as well, and it would conflict with `Cust_XXXX.msi`
4. Install `Cust_XXXX.msi` needed:
